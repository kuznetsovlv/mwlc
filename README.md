## Utils
### 2absp
Converts file pathes to absolute. NOTE: 2absp **DOES NOT** check if pathes exist.
#### Install
As root:
```bash
    make install_2absp
```
#### Usage
```bash
    2absp PATH...
    2absp OPTION
```
See man.
